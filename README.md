## Spam Email Classification using Scikit-learn
 # Project Overview

This project implements a Machine Learning classification model using Scikit-learn to detect whether a message is Spam or Not Spam.
The model is built using:
1. Text vectorization (CountVectorizer)
2. Naive Bayes classification (MultinomialNB)
3. Train-test split with stratification
4. Model evaluation using accuracy and classification report

This project demonstrates the complete machine learning workflow:
data preparation → feature extraction → model training → evaluation → prediction.

# Technologies Used

1. Python 3.x
2. Pandas
3. Scikit-learn

# Installation
1️. Install Python

Ensure Python 3 is installed.

Check version:

python --version

2️. Install Required Libraries:
pip install pandas scikit-learn

# How to Run the Project

Download or clone the repository.
Open terminal/command prompt in the project folder.

Run:
    python spam_classifier.py

The program will:
1. Train the model
2. Display accuracy
3. Show classification report
4. Predict a new email

These metrics evaluate how well the model performs on unseen data.
